💬 通讯联系
Meet
Google Meet
与 Cindy 视频通话（澳大利亚联系用）
  1. 点开绿色摄像机的 Meet 图标
  2. 点"新建通话"
  3. 输入或选择 Cindy 的联系方式
  4. 点击通话按钮开始视频
Cindy 发来 Meet 链接时，直接点链接就能加入通话
小紫人会议
Microsoft Teams
和 Cindy 视频通话、发消息
  1. 打开 Teams，点击 聊天 找到 Cindy
  2. 点击 Cindy 的名字，再点 📹 视频通话 或 📞 语音通话
  3. 也可以直接发文字消息给 Cindy
用 Teams 联系 Cindy，不需要手机信号，只需要 WiFi
🚌 出行交通
MetroTas
MetroTas
查询金斯顿巴士时刻表和路线
  1. 点开绿色 M 字母的 MetroTas 图标
  2. 选择"出发地"和"目的地"
  3. 查看下一班巴士的时间和站台
  4. 上车后记得用 Greencard 刷卡
如果不确定，可以问 Cindy 帮你查路线
Transit
Transit
公共交通导航，查询巴士实时位置
  1. 点开绿色 Transit 图标
  2. 打开后自动显示附近的巴士站
  3. 选择要去的目的地
  4. 查看最近一班巴士的到站时间
可以看到巴士在地图上的实时位置
地图
Google Maps
查找地址、步行/驾车导航
  1. 点开有地图标志的 Google 地图图标
  2. 在搜索框输入目的地名称
  3. 选择步行🚶或驾车🚗模式
  4. 点"开始导航"，跟着蓝色箭头走
可以用中文搜索地址，例如："Woolworths Kingston"
🛒 购物与银行
ANZ
ANZ Bank
查看银行账户余额、账单记录
  1. 点开黑色 ANZ 图标
  2. 用指纹或密码登录
  3. 可以看到账户余额
  4. 点账户名称查看最近交易记录
只能查看，不用担心操作错误。如有疑问请找 Cindy
钱包
Google Wallet
用手机付款（刷手机代替刷卡）
  1. 在超市结账时，告诉收银员"手机付款"
  2. 将手机背面靠近收款机
  3. 手机屏幕亮起并显示绿色对勾即表示付款成功
不用解锁手机，直接靠近机器就能付款。如有问题请找 Cindy
🎬 娱乐休闲
YouTube
YouTube
看视频、中文节目、新闻、教程，免费
  1. 点开红色 YouTube 图标
  2. 点上方搜索框，输入想看的内容
  3. 例如：搜"中文新闻"或"太极拳教程"
  4. 点视频名称开始播放
  5. 屏幕向横放可以全屏观看
不需要账号也可以看大部分视频
TikTok
TikTok / 抖音国际版
刷短视频娱乐，有很多有趣的中文内容
  1. 点开黑色 TikTok 图标
  2. 打开自动播放推荐视频
  3. 向上滑动看下一个视频
  4. 点心形图标表示喜欢
系统会根据你的喜好推荐更多你感兴趣的内容
相册
Google Photos
查看手机里的所有照片和视频
  1. 点开彩色风车图案的相册图标
  2. 可以看到所有拍过的照片
  3. 点照片可以放大查看
  4. 两根手指撑开屏幕可以进一步放大
  5. 向左右滑动可以翻看其他照片
照片自动备份到云端，不会丢失
📚 学习与翻译
Drops
Drops
用游戏方式学英语单词，每天5分钟
  1. 点开紫色水滴的 Drops 图标
  2. 选择想学的主题（如：食物、交通）
  3. 看图片猜英语单词，拖动匹配
  4. 每天玩5分钟，坚持积累词汇
免费版每天可以玩5分钟，很适合每天练习
Quizlet
Quizlet
英语单词卡片学习，Cindy 制作了专属单词表
  1. 点开蓝色 Quizlet 图标
  2. 找到 Cindy 为你准备的单词表
  3. 点"学习"开始练习
  4. 看中文猜英文，或看英文猜中文
Cindy 每周会添加新单词，一起学英语！
沉浸式翻...
Immersive Translate
在浏览网页时自动将英文翻译成中文
  1. 用 Chrome 浏览器打开英文网页
  2. 点屏幕下方的翻译按钮
  3. 英文段落下方会出现中文翻译
特别适合看英文信件、账单时使用
Hi Transl...
Hi Translate
拍照翻译、文字翻译，中英文互译
  1. 点开 Hi Translate 图标
  2. 在输入框输入中文 → 看英文翻译
  3. 或者输入英文 → 看中文翻译
  4. 也可以点相机图标，拍照翻译菜单或标牌
在超市看不懂英文标签时，用拍照翻译功能很方便
🏠 家庭与安全
Life360
Life360
家庭位置共享，Cindy 可以看到你们在哪里
  1. 这个 App 只需要保持打开状态就好
  2. Cindy 可以在她的手机上看到你们的位置
  3. 如果你们出门了，Cindy 知道你们平安
  4. 不需要主动操作，保持后台运行即可
这个 App 是为了安全，Cindy 在担心时可以确认你们平安
OneDrive
Microsoft OneDrive
Cindy 存放重要文件的地方，签证和文件都在这里
  1. 这个 App 主要由 Cindy 使用和管理
  2. 如果 Cindy 让你发某个文件，在这里找
  3. 点蓝色云朵图标打开
  4. 找到对应文件夹，点文件查看或发送
签证、保险等重要文件都保存在这里，不要删除任何文件
